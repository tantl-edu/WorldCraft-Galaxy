# WorldCraft Galaxy

WorldCraft Galaxy is a web-first block-building exploration game prototype. Players build fictional planets, gather real and invented resources, swap materials with other worlds, and eventually explore a larger galaxy.

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm test
```

## First Prototype

- Fictional starter planet: Auralis Reach
- 3D block building scene
- Build palette with real and invented resources
- Inventory and resource values
- Swap / Trade panel with pending trade offers
- GitHub Pages deployment workflow
